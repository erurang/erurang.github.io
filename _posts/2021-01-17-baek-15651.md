[15651 : N과M(3)](https://www.acmicpc.net/problem/15651)

## 백트랙킹 문제다.

제출코드

```
a,b = map(int,input().split())

tc = [str(n) for n in range(1,a+1)]

def back(index,path):
    
    # 만약에 숫자길이와 지금까지 더한 숫자길이가 같다면
    if b == len(path):
        for i in path:
            print(i,end=" ")
        print("")
        return

    for i in range(index,len(tc)):
        for j in tc[i]:
            back(index,path+j)

    return

back(0,"")

```
