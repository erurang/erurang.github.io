## Next.js

일단 Node.js가 기본적으로 깔려있어야합니다!!

`npm init` 으로 pacakge.json을 만들어준다.
```
"scripts": {
    "dev": "next"
  },
```
를 위와같이 수정해준다.

`npm i next`

Next.js는 react의 프레임워크로 SSR방식을 가능하게 도와줍니다.

[이곳](https://erurang.github.io/web/2021/01/19/web-base-ssrcsr/)에 CSR과 SSR의 차이를 정리해두었습니다.
